# Dataset and model licenses

No dataset has been downloaded in phase 1.

- M4Singer: official project <https://github.com/M4Singer/M4Singer>; described
  by its maintainers as non-commercial research data. Review and preserve the
  license/citation shipped with the downloaded archive before use.
- OpenCpop: obtain only through <https://wenet-e2e.github.io/opencpop/download/>.
- MIR-MLPop: <https://github.com/york135/MIRMLPop>; review its academic,
  non-commercial terms before use.
- Whisper: <https://github.com/openai/whisper>; preserve the model license.

This file is a provenance checklist, not legal advice. Missing or inaccessible
datasets must be reported as unavailable and never replaced with unknown copies.
